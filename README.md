# Next.js Developer Portfolio

A modern, responsive **developer portfolio website** built with **Next.js** to showcase my projects, skills, and experience. Designed with performance, accessibility, and clean UI in mind.

---

## Live Demo

**Portfolio Website:**  
https://my-portfolio-lemon-two-94.vercel.app/

---

## Tech Stack

- **Framework:** Next.js (App Router / Pages Router)
- **Language:** JavaScript / TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide / React Icons
- **Deployment:** Vercel

---

## Features

- Fast and SEO-friendly (Next.js)
- Fully responsive design (mobile-first)
- Clean UI with modern animations
- Projects showcase with live/demo links
- About & Skills section
- Contact form / social links
- Dark mode support
